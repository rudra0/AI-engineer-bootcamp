# AI Engineer Bootcamp

Repository connected to the local AI Engineer Bootcamp project.

Each folder will contain learning scripts and documentation to make this repository easy to follow for other learners. The `Python` folder already includes dedicated markdown files for each script, and the new `Numpy-Pandas` folder includes NumPy and pandas examples along with practice problems and reference solutions.

Folders:
- Python
- Numpy-Pandas
- Statistics
- Machine-Learning
- Deep-Learning
- Transformers
- LLMs
- RAG
- AI-Agents
- MLOps
- System-Design

Numpy-Pandas includes:
- `Numpy-Pandas/numpy_basics/` — NumPy array creation, shapes, dtype, and reductions
- `Numpy-Pandas/numpy_array_ops/` — arithmetic, broadcasting, slicing, and stacking examples
- `Numpy-Pandas/pandas_basics/` — DataFrame creation, selection, and summary operations
- `Numpy-Pandas/pandas_groupby/` — grouping and aggregation examples
- `Numpy-Pandas/problems/` — practice problem descriptions for NumPy and pandas
- `Numpy-Pandas/solutions/` — runnable solution scripts for practice problems
