# AI Engineer Bootcamp

Repository connected to the local AI Engineer Bootcamp project.

Each folder will contain learning scripts and documentation to make this repository easy to follow for other learners. The `Python` folder already includes dedicated markdown files for each script, and the new `Numpy-Pandas` folder includes NumPy and pandas examples along with practice problems and reference solutions.

Folders:
- Python
- Numpy-Pandas
- Statistics
- Machine-Learning
- Deep-Learning
- Transformers
- LLMs
- RAG
- AI-Agents
- MLOps
- System-Design
