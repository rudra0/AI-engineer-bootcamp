# Banking System Using Functions

## Purpose

A simple banking application built using procedural programming.

## What it teaches

- using loops for menus
- handling user input
- deposit and withdrawal operations
- validating available balance
- exiting a repeating menu cleanly

## How to run

```bash
python3 bankingSystemUsingFunction.py
```

## Example

- Choose `1` to deposit money
- Choose `2` to withdraw money
- Choose `3` to view the current balance
- Choose `4` to exit the program
