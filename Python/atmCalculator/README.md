# ATM Calculator

## Purpose

This script simulates a very basic ATM interface for deposit, withdrawal, and balance checking.

## What it teaches

- `input()` usage
- `if` / `elif` / `else` branching
- numeric conversion with `float()`
- simple arithmetic operations

## How it works

1. It starts with a fixed `balance` value.
2. It asks the user to choose an option: deposit, withdraw, or check balance.
3. Depending on the choice, it updates the balance or prints the current balance.

## Run

```bash
python3 atmCalculator.py
```

## Example

```text
Enter the selection number: 1. Deposit 2. Withdraw 3. Check Balance: 1
Enter the amount to deposit: 100
Deposit successful. Your new balance is: 1100.0
```
