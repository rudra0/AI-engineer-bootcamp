# Grade Calculator

## Purpose

Convert numeric grades into letter grades and demonstrate branching logic.

## What it teaches

- conditionals with multiple ranges
- user input conversion to `float`
- formatted string output using f-strings

## How it works

1. The script reads the user's name and numeric grade.
2. It checks the grade value against multiple thresholds.
3. It prints the corresponding letter grade.

## Run

```bash
python3 gradeCalculator.py
```

## Example

```text
Enter your name: Alice
Enter your grade: 88
Alice, you got a B!
```
