# Pattern Printing

## Purpose

Print a simple star pattern to demonstrate loops and string multiplication.

## What it teaches

- using `for` loops with `range()`
- repeating characters with string multiplication
- output formatting

## How it works

1. The script loops from 1 to 5.
2. For each iteration, it prints that many stars (`*`).

## Run

```bash
python3 pattern.py
```

## Example Output

```text
*
**
***
****
*****
```
