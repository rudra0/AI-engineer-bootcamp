# Student Management System Pro

## Project features

This project will implement the following features:

- ✅ Add Student
- ✅ View Students
- ✅ Search Student
- ✅ Delete Student
- ✅ Save Data
- ✅ Load Data
- ✅ OOP
- ✅ File Handling
- ✅ Exception Handling
- ✅ JSON Storage

## Folder structure

- `main.py` — application entry point
- `students.json` — JSON storage for student data

## Next steps

1. Implement student classes and data handling in `main.py`.
2. Add JSON save/load functionality using `students.json`.
3. Add exception handling for invalid input and file errors.
4. Add search, delete, and view features.

## Run

From the `Python/student_management/` folder:

```bash
python3 main.py
```

## Menu options

- `1` Add Student
- `2` View Students
- `3` Search Student
- `4` Delete Student
- `5` Save
- `6` Exit

The application loads students from `students.json` automatically at startup, and you can save changes any time with option `5` or by exiting.
