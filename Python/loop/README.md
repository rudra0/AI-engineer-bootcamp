# Loop Examples

## Purpose

Demonstrate basic `for` loop usage in Python.

## What it teaches

- iterating over a range of numbers
- using the modulus operator `%` to find even numbers
- creating a multiplication table with loops

## How it works

1. It prints even numbers between 1 and 10.
2. It asks the user for a number and prints its multiplication table from 1 to 10.

## Run

```bash
python3 loop.py
```

## Example

```text
1 is not printed
2 is even
...
Enter a number to see its multiplication table: 5
5 x 1 = 5
5 x 2 = 10
...
```
