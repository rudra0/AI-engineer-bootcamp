# Recursion

## Purpose

Understand how recursive functions call themselves to solve problems.

## What it teaches

- recursive function design
- base cases
- stack behavior
- computing factorial and Fibonacci numbers

## How it works

1. `factorial()` calls itself with `n-1` until it reaches 1.
2. `fibonacci()` calls itself twice for smaller values.
3. The script prints factorial and a Fibonacci sequence.

## Run

```bash
python3 recursion.py
```
