# Comprehensions

## Purpose

Learn list, dictionary, and set comprehensions for concise Python expressions.

## What it teaches

- list comprehensions
- dictionary comprehensions
- set comprehensions
- filtering values inside comprehensions

## How it works

1. Builds a list of squares.
2. Filters even numbers from a range.
3. Creates a dictionary of word lengths.
4. Builds a set of unique characters.

## Run

```bash
python3 comprehensions.py
```
