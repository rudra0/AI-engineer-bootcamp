# Password Strength Checker

## Purpose

Check whether a password meets basic strength rules.

## What it teaches

- string length validation
- checking for uppercase letters
- checking for lowercase letters
- checking for digits
- checking for special characters

## How to run

```bash
python3 passwordStrengthChecker.py
```

## Example

- Password too short: shows a length requirement message
- Missing uppercase: asks for at least one uppercase letter
- Missing symbol: asks for a special character
- Strong password: prints `Password is strong.`
