# File I/O

## Purpose

Learn how to read from and write to files in Python.

## What it teaches

- creating and writing text files
- reading file contents
- working with the `pathlib` module

## How it works

1. `write_sample_data()` creates or overwrites `sample_data.txt`.
2. `read_sample_data()` reads the file back into Python.
3. The script prints the file lines to the terminal.

## Run

```bash
python3 fileIO_example.py
```
