# Student Record Management System

## Topic

Classes and object-oriented programming in Python.

## Exercise

Build a simple student record manager using Python classes. The system should:

- create student objects with `name`, `age`, and `grade`
- store students in a list
- display all student records
- search for a student by name

## How to run

```bash
python3 studentRecordManagementSystem.py
```

## What you learn

- defining a class with `__init__`
- working with object instances
- implementing search logic
- structuring code for reuse and readability
