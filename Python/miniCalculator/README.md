# Mini Calculator

## Purpose

Perform basic arithmetic operations between two numbers.

## What it teaches

- reading numeric input as strings
- converting input to `float`
- using conditional branching to select an operation
- printing formatted math results

## How it works

1. The script reads two input values.
2. It asks the user to select add, subtract, multiply, or divide.
3. It performs the chosen operation and prints the result.

## Run

```bash
python3 miniCalculator.py
```

## Example

```text
Enter first number: 8
Enter second number: 2
Select operation.
1.Add
2.Subtract
3.Multiply
4.Divide
Enter choice(1/2/3/4): 4
8 / 2 = 4.0
```
