# Generators

## Purpose

Learn how Python generators produce values lazily using `yield`.

## What it teaches

- creating generator functions
- using `yield`
- iterating over generator output
- avoiding large memory use for sequences

## How it works

1. `even_numbers()` yields even numbers up to a limit.
2. The script iterates over the generator and prints each value.

## Run

```bash
python3 generators.py
```
