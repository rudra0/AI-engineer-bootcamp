# Inheritance

## Purpose

Learn how to use class inheritance in Python to reuse and extend behavior.

## What it teaches

- defining a base class
- creating subclasses
- overriding methods
- using polymorphism

## How it works

1. `Animal` is the base class.
2. `Dog` and `Cat` inherit from `Animal`.
3. Each subclass overrides the `speak()` method.
4. The script prints each animal's sound.

## Run

```bash
python3 inheritance.py
```
