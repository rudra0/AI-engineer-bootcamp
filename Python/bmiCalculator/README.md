# BMI Calculator

## Purpose

This script calculates Body Mass Index (BMI) from a user-provided weight and height.

## What it teaches

- receiving numeric input from the user
- performing exponentiation with `**`
- basic formula implementation and printing results

## How it works

1. It reads weight in kilograms and height in meters.
2. It computes BMI using the formula `weight / (height ** 2)`.
3. It prints the resulting BMI.

## Run

```bash
python3 bmiCalculator.py
```

## Example

```text
Enter your weight in kg: 70
Enter your height in meters: 1.75
Your BMI is: 22.857142857142858
```
