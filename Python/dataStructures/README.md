# Data Structures

## Purpose

Learn basic Python data structures: lists, tuples, sets, and dictionaries.

## What it teaches

- creating and modifying lists
- using tuples for fixed data
- deduplicating values with sets
- storing key-value data with dictionaries

## How it works

1. `list_example()` adds items to a list.
2. `tuple_example()` shows an immutable tuple.
3. `set_example()` demonstrates unique collection behavior.
4. `dict_example()` stores and retrieves inventory data.

## Run

```bash
python3 dataStructures.py
```
