# Exception Handling

## Purpose

Learn how to handle runtime errors and protect Python programs from invalid input.

## What it teaches

- using `try` / `except`
- creating custom exceptions
- validating input safely
- handling division by zero

## How it works

1. `parse_positive_int()` converts strings to positive integers.
2. It raises helpful exceptions for invalid or negative values.
3. `safe_divide()` handles division by zero gracefully.

## Run

```bash
python3 exceptionHandling.py
```
