# Modules

## Purpose

Learn how to split code into reusable modules and import functions.

## What it teaches

- creating a helper module
- importing functions
- keeping code organized

## How it works

1. `utils.py` defines a reusable `greet()` function.
2. `main.py` imports `greet()` and calls it.
3. This structure demonstrates modular Python code.

## Run

```bash
cd modules
python3 main.py
```
