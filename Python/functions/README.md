# Functions

## Purpose

Learn how to define and use functions in Python.

## What it teaches

- creating functions with parameters
- using default arguments
- returning values from functions
- organizing reusable code

## How it works

1. `greet()` prints a welcome message.
2. `calculate_average()` returns the mean of a list of numbers.
3. `factorial()` computes the factorial of a positive integer.

## Run

```bash
python3 functions.py
```
