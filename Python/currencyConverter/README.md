# Currency Converter

## Purpose

Convert a USD amount into another currency using fixed conversion rates.

## What it teaches

- user choice handling with `input()`
- `if` / `elif` / `else` statements
- performing currency conversion calculations

## How it works

1. The script asks for an amount in USD.
2. The user selects the target currency: INR, EUR, or GBP.
3. It prints the converted value using a fixed exchange rate.

## Run

```bash
python3 currencyConverter.py
```

## Example

```text
Enter amount in USD: 50
Select currency to convert to:
1. INR
2. EUR
3. GBP
Enter choice(1/2/3): 2
50.0 USD is equal to 42.5 EUR
```
