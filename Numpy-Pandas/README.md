# Numpy-Pandas

This folder contains hands-on NumPy and pandas examples, problem statements, and reference solutions similar in style to the `Python` folder.

Structure:

- `numpy_basics/` - simple NumPy introductions
- `numpy_array_ops/` - array operations and examples
- `pandas_basics/` - DataFrame creation and basic ops
- `pandas_groupby/` - grouping and aggregation examples
- `problems/` - problem statements for practice
- `solutions/` - runnable Python solutions for the problems

Run examples with `python3 <script>.py`.
